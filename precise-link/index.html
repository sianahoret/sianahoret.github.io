Would you like to be able to MAKE A HYPERLINK TO ANY TEXT ON A PAGE?


Currently browsers resolve a reference to a specific place of a web page (by a "fragment identifier" in a URL) only if the page contains a corresponding mark (either an anchor tag or an element id/name attribute). However you may want to link to an arbitrary part of a web page.

Here are two very simple bookmarklets allowing you to save or share a link to an arbitrary phrase on a page so that later it will be possible to open the link and to see exactly the referred place shown and selected.

HOW TO USE

* Below "precise link" is a URL made with "Make Precise Link" bookmarklet.

Making a "precise link"

  1. Now (once and forever) drag and drop Make Precise Link bookmarket into bookmarks in your browser.
  2. On any web page when want to link to some text, select the text. The text should be as possible unique in the scope of the page.
  3. Click "Make Precise Link" bookmark created on the step #1.
  4. A pop-up will appear. Copy the hyperlink there.
  5. Save or share the hyperlink.

Using a "precise link"

  1. Now (once and forever) drag and drop Resolve Precise Link bookmarklet into bookmarks in your browser.
  2. Visit a web page by a "precise link" (which you have obtained from somebody or saved by yourself earlier).
  3. Click "Resolve Precise Link" bookmark.

Additional notes about a "precise link":

  • It is the target page (where "Make Precise Link" bookmark was applied to) URL appended with an additional hash-parameter (#href-to-text=thetext)
  • It does not affect the shown page content and behaviour anyway
  • When you visit the target page by a "precise link" and a browser does not have "Resolve PreciseLink" bookmark (or you do not click the bookmark), thepage will be shown exactly as if it was visited by its original URL.
  • it is not processed by any additional web service

Alternatives
